3.1 Promjena svjetline LED diode bez vanjskog upravljanja

 

Digitalnim pinovima 3, 5, 6, 9, 10 i 11  možemo upravljati svjetlinom LED diode, brzinom vrtnje DC motora i nekim drugim stvarima koristeći funkciju analoglWrite().

Pošto su ti pinovi digitalni to znači da mogu biti na naponu ili 0V ili 5V . To je digitalni signal.

(NIKAKO VRIJEDNOSTI IZMEĐU 0V i 5V ) Naziv takvog signala je „Pulsno širinska modulacija“ koja se naširoko koristi u uređajima poput mobitela, laptopa, tableta, TV-a, kućnog računala, pametnim žaruljama, robotima itd. (Skraćeno engl. PWM od „Pulse width modulation“)

 

Karakteristike PWM-a na Arduinu.

 

PWM ima STALNU FREKVENCIJU.

 

Mijenja se samo popunjenost perioda  naponom od 5V u cjelobrojnim koracima od 0 do 255. ( 8 bitni signal jer 28 = 256 – 1).

 

PRIMJERI:

Ako je popunjenost perioda PWM signala naponom od 5V u periodu

 0% - srednji napon je USR = 0 V

25% - srednji napon je USR = 1.25 V

50% - srednji napon je USR = 2.5 V

75% - srednji napon je USR = 3.75 V

100% - srednji napon je USR = 5 V

Upišite sljedeći program.

 

int svjetlina = 0;    

int fade = 5;    

void setup() {

  pinMode(9, OUTPUT);

}

void loop() {  

  analogWrite(9, svjetlina);  

  svjetlina = svjetlina + fade;

  if (svjetlina <= 0 || svjetlina >= 255) {

    fade = -fade;

  }

  delay(30);

}

 

LED dioda bi trebala mijenjati svjetlinu.

 

3.2 Upravljanje svjetlinom LED diode potenciometrom

 

Samostalno mijenjanje svjetline LED diode ima smisao ako se radi npr. o signalizaciji na mobitelu da smo primili poruku, e-mail itd.

U praktičnoj uporabi nema smisla ako mi ne možemo upravljati svjetlinom, npr. rasvjeta u sobi.

Današnje pametne žarulje najčešće koriste daljinske upravljače. Mi ćemo koristiti ono što već znamo, potenciometar spojen na analogni pin i ADC.

Ovdje se pojavljuje problem nekompatibilnosti ADC-a i PWM-a.

ADC je 10 bitni, a PWM 8 bitni. Zato ćemo koristiti funkciju map() koja jedan raspon brojeva pretvara u drugi tj. 10 bitni u 8 bitni tj. raspon brojeva od  0-1023 u raspon od 0-255 .

(Funkcija map() podržava i druge raspone brojeva ovisno o potrebi programera.)

 

Upišite sljedeći program.

 

void setup() {

  pinMode(3, OUTPUT);

}

void loop() {  

  int _10bitno = analogRead (A0);

  int _8bitno = map(_10bitno ,0,1023,0,255);

  analogWrite(3, _8bitno);  

  delay(30);

}

 

Ugađanjem potenciometra mijenjamo svjetlinu LED diode.

 

 

analogWrite() generira PWM -  pulsno širinsku modulaciju

Sintaksa: analogWrite(pin, vrijednost)

Argumenti: pin je broj pina, vrijednost su cijeli brojevi od 0 do 255 (28=256-1)

Razvojna pločica: Arduino Uno

Pinovi: 3, 5, 6, 9, 10, 11

Frekvencija: 490 Hz (pinovi 5 i 6: 980 Hz)

 

map() preslikava niz brojeva iz jednog raspona u drugi (linearno)

Sintaksa: map(varijabla, od najmanjeg, od najvećeg, u najmanji , u najveći)

Primjer: Imamo niz vrijednosti od 0 do 100. Želimo pretvoriti taj niz brojeva u niz od 25 do 75.

Tada pišemo:   map(varijabla, 0, 100, 25, 75)

Sada je vrijednost 0 postala 25 a vrijednost 100 je postala 75 i ostalo sve između je linearno preslikano.

Primjer:

VARIJABLA = map(VARIJABLA, od najmanjeg, od najvećeg, u najmanji , u najveći)

Funkcija vraća vrijednost !!!
